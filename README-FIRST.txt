TiHiY STREAMCONTROL CENTER — ТЕСТОВА ВЕРСІЯ CHAT CORE

ЩО ВЖЕ Є:
- чат і журнал подій у самій програмі;
- тестові Twitch/YouTube повідомлення;
- команди бота;
- окремий overlay;
- overlay закривається разом із програмою;
- прозорість змінює тільки фон;
- текст чату лишається чітким;
- пропуск кліків вмикається і вимикається в головній програмі;
- внизу overlay показуються глядачі та лайки.

ЯК ЗІБРАТИ ГОТОВИЙ EXE:
1. Встановити .NET 8 SDK.
2. Запустити BUILD-RELEASE.bat.
3. Готовий EXE з'явиться у папці Release.

ВАЖЛИВО:
Це основа Chat Core. Реальні Twitch/YouTube API, OAuth, алерти та OBS будуть наступними модулями.

ЯК ЗАВАНТАЖИТИ В GITHUB ОДНІЄЮ КНОПКОЮ:
1. Розпакувати цей один ZIP.
2. Запустити UPLOAD-TO-GITHUB.bat.
3. Підтвердити вхід у GitHub у браузері, якщо з'явиться запит.
