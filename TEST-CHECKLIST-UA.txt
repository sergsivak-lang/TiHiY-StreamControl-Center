TIHIY STREAMCONTROL CENTER v0.9.0 — ПОВНИЙ ЧЕКЛИСТ ТЕСТУ
========================================================

1. ЗАПУСК І ЗАВЕРШЕННЯ
[ ] BUILD-AND-RUN.bat завершується написом BUILD OK.
[ ] Відкривається тільки одна копія програми.
[ ] Повторний запуск показує повідомлення, що програма вже працює.
[ ] Після закриття хрестиком TiHiY.StreamControlCenter.exe зникає з Диспетчера задач.

2. ІНТЕРФЕЙС І ТЕМИ
[ ] Головне вікно має темний Cyber Amber фон без світлої сітки.
[ ] Усі кнопки мають векторні іконки та HUD-рамки.
[ ] Вікно можна масштабувати кнопками − / АВТО / +.
[ ] Вертикальний і горизонтальний розділювачі змінюють розміри блоків.
[ ] Налаштування → Теми змінює все головне вікно і вже відкриті модулі.
[ ] Після перезапуску зберігається вибрана тема, масштаб і розміри блоків.

3. OBS AUDIO
[ ] OBS → Налаштування → WebSocket Server увімкнено на 127.0.0.1:4455.
[ ] Пароль збережено і не злітає після перезапуску.
[ ] «ПІДКЛЮЧИТИ OBS» змінює статус на ПІДКЛЮЧЕНО.
[ ] Швидкий мікшер показує до 6 вибраних джерел.
[ ] Рівні сигналу рухаються синхронно з OBS Audio Mixer.
[ ] Повзунок у програмі змінює гучність в OBS.
[ ] Зміна гучності в OBS оновлює повзунок у програмі.
[ ] MUTE синхронізується в обидва боки.
[ ] Повний мікшер прокручується і дозволяє закріпити потрібні джерела.

4. TWITCH
[ ] Канали → Twitch: канал, Client ID і Client Secret заповнені.
[ ] Авторизація відкривається в браузері.
[ ] Статус Twitch стає ПІДКЛЮЧЕНО.
[ ] Повідомлення Twitch з'являються в мультичаті з фіолетовою іконкою.
[ ] Кнопка TWITCH надсилає повідомлення в Twitch чат.
[ ] Модерація тайм-аут/бан/видалення працює з реальним повідомленням.

5. YOUTUBE
[ ] Канали → YouTube: Google OAuth Client ID і Client Secret заповнені.
[ ] Авторизація відкривається в браузері без помилки 403.
[ ] Статус YouTube стає ПІДКЛЮЧЕНО.
[ ] Повідомлення YouTube з'являються з червоною іконкою.
[ ] Кнопка YOUTUBE надсилає повідомлення в YouTube чат.
[ ] YouTube Studio відкриває:
    https://studio.youtube.com/channel/UC4-t_7-LD_E15LXazQmsq_g/livestreaming/dashboard
[ ] Вікно трансляції завантажує список ефірів та зберігає зміни.

6. DISCORD BOT
[ ] Bot Token зберігається після закриття програми.
[ ] Бота додано на сервер із правом Send Messages.
[ ] Окремо задано ID каналу старту стрімів.
[ ] Окремо задано ID каналу донатів і платних підписок.
[ ] Тестові повідомлення приходять у правильні канали.
[ ] При старті Twitch/YouTube бот надсилає посилання на трансляцію.

7. DONATELLO
[ ] Введено офіційний API X-Token.
[ ] «Перевірити токен» показує профіль і статус ПІДКЛЮЧЕНО.
[ ] Історія донатів завантажується з /api/v1/donates.
[ ] Платні підписки завантажуються з /api/v1/subscribers.
[ ] Старі записи не відправляються повторно в overlay/Discord/чат.
[ ] Новий донат з'являється у головній панелі, журналі й overlay.
[ ] Повтор вибраної події повторює тільки overlay, без дубля в чатах.
[ ] Сповіщення монетизації приходить в окремий Discord-канал.

8. OVERLAY
[ ] OBS Browser Source відкриває URL із модуля Overlay.
[ ] Чат не мерехтить.
[ ] У повідомлень немає часових міток.
[ ] Фон прозорий або має вибрану прозорість.
[ ] Платформу показано іконкою, а не написом.
[ ] «ВІДКРИТИ ЧАТ ПОВЕРХ ГРИ» показує локальне прозоре вікно.
[ ] Click-through дозволяє керувати грою крізь overlay.

9. МУЗИКА / NOW PLAYING
[ ] Додавання файлів і папки працює.
[ ] Play/Pause/Next/Previous, гучність, repeat і shuffle працюють.
[ ] Плейлист відновлюється після перезапуску.
[ ] Now Playing Browser Source показує актуальний трек.

10. ЖУРНАЛ
[ ] ЖУРНАЛ ПОДІЙ відкриває реальне вікно журналу.
[ ] Пошук фільтрує записи.
[ ] Копіювання виділеного/всього працює.
[ ] «Відкрити папку логів» відкриває %APPDATA%\TiHiY\StreamControlCenter\Logs.
