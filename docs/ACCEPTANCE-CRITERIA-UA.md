# Критерії приймання Cyber Amber

## Обов’язковий вигляд

Еталон: `Cyber-Amber-TARGET.png`.

Тема має змінювати не тільки кольори, а всю систему відображення:

- власний заголовок вікна та кнопки керування;
- HUD-панелі зі скошеними кутами й локальними декоративними лініями;
- нові шаблони кнопок та тематичні векторні іконки;
- нові TextBox, ComboBox, CheckBox, Slider, ProgressBar, ScrollBar і статуси;
- аудіомікшер із сегментованими індикаторами рівня;
- однаковий стиль головного та всіх дочірніх вікон;
- без розтягнутого фонового зображення з білими лініями;
- адаптація до зміни розміру вікна без накладання кнопок і тексту.

## Обов’язкова функціональність

- захист від запуску кількох копій;
- повне завершення процесу після закриття;
- OBS WebSocket: джерела звуку, гучність, MUTE, двостороння синхронізація;
- Twitch і YouTube OAuth та чат;
- Discord Bot і окремі канали сповіщень;
- офіційний Donatello API через `X-Token`;
- чат overlay без часових міток, без миготіння, з іконками платформ;
- журнал подій;
- музика й Now Playing.

## Умови готовності

1. GitHub Actions `Build Windows Portable` завершується успішно.
2. Startup smoke test проходить.
3. На Windows зроблено реальний скріншот через `VERIFY-LOCAL.bat`.
4. Скріншот візуально порівняно з еталоном.
5. Пройдений `TEST-CHECKLIST-UA.txt`.
