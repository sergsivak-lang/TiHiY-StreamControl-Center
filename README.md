# TiHiY StreamControl Center v0.4

Перша демонстраційна версія окремого центру керування стрімом для TiHiY-DED.

## Уже працює

- головне WPF-вікно в стилі Star Citizen MFD + Minimal Ukrainian;
- демонстраційний об’єднаний чат;
- надсилання локальних тестових повідомлень;
- журнал подій;
- тестовий донат Donatello;
- екран команд бота;
- прозорий чат-overlay поверх усіх вікон;
- зміна прозорості, переміщення та режим пропускання кліків;
- демонстраційні кнопки запуску/завершення стріму.
- компактні статуси сервісів: зелений `ON` або червоний `OFF`;
- виправлена примусова темна тема та контраст тексту.
- нижній моніторинг трансляції: якість, час, бітрейт, FPS, пропущені кадри та втрата пакетів Twitch/YouTube.
- реальне OBS WebSocket 5.x підключення без сторонніх бібліотек;
- запуск/зупинка трансляції та читання статистики OBS;
- локальне збереження адреси OBS; пароль навмисно не записується на диск;
- self-contained single-file збірка Windows x64.
- глобальні гарячі клавіші overlay: `Ctrl+Shift+C` показ/приховування, `Ctrl+Shift+L` пропускання кліків;
- збереження прозорості overlay між запусками.
- власна темна рамка вікна та MFD-заголовок;
- стабільні темні кнопки з hover/pressed-станами без підміни системною темою Windows.
- робочий редактор команд бота: створення, зміна, видалення, cooldown, платформи та імпорт JSON/CSV;
- локальне виконання команд у чаті та автоматичні відповіді TiHiY-BOT;
- робочі сторінки меню «Алерти», «Overlay», «OBS» і «Донати» з тестами та збереженням параметрів.

## Запуск

1. Встановити **.NET 8 SDK x64**.
2. Розпакувати проєкт.
3. Запустити `build.bat`.

Або відкрити `TiHiY.StreamControlCenter.csproj` у Visual Studio 2022 і натиснути F5.

## Важливо

OBS WebSocket уже підключається напряму до локального OBS Studio. Twitch, YouTube і Donatello поки працюють у демонстраційному режимі та потребують окремої безпечної авторизації без збереження секретів у репозиторії.
